Easy to use Wordpress plugin to insert Fellowship Stream MyBible to your posts, pages and sidebar.

== Description ==
Use this plugin to quickly and easily insert Fellowship Stream MyBible into your posts, pages and sidebar by using a trigger text or calling the php function.

MyBible allows your web site visitors to read complete versions of the Bible on your site without being redirected to a third party site.

MyBible is a commercial application sold by Fellowship Stream.  You will be able to rename it and customize it to fit your site.

For information and updates, please visit:
http://www.fellowshipstream.com/index.php?option=com_content&view=section&layout=blog&id=4&Itemid=17

Usage:

There are two ways you can use this plugin:

   1. Add the trigger text 
         <!-- wp_fs_myb_1 -->
      or <!-- wp_fs_myb_2 -->
      or <!-- wp_fs_myb_3 --> to your posts or pages
   2. Call the function from template files:
         <?php echo show_fs_myb_1(); ?>
      or <?php echo show_fs_myb_2(); ?>
      or <?php echo show_fs_myb_3(); ?>


== Installation ==

1. Unzip and Upload the folder 'WP-FellowshipStream-MyBible' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings and configure the options eg. Copy and paste the Fellowship Stream MyBible code that you want to use.
4. Add the trigger text <!-- wp_fs_myb_1 --> to a post or page where u want it to appear.
